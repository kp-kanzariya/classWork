Execution:

    call and apply execute the function immediately.
    bind returns a new function that can be invoked later.
    Arguments Handling:

    call takes arguments as a list.
    apply takes arguments as an array.
    bind takes initial arguments that are prepended to the arguments provided when the new function is called.

    call: Calls a function with a specified this context and individual arguments.
    bind: Returns a new function with a specified this context and initial arguments.
    apply: Calls a function with a specified this context and arguments provided as an array.
